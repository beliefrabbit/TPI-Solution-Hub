# TPI Solution Hub

一個現代化的解決方案展示平台，用於管理和展示成功案例、技術解決方案和業務成果。

## 功能特色

- 🎨 **現代化 UI 設計** - 粒子動畫背景和玻璃態效果
- 📊 **案例管理系統** - 完整的後台管理介面
- 🏷️ **標籤分類系統** - 支援產業別、業務別、趨勢科技三種分類
- 🔍 **智能搜尋** - AI 驅動的案例搜尋功能
- 📤 **Excel 匯入/匯出** - 支援批量匯入和匯出案例資料
- 🖼️ **圖片上傳** - 支援本地上傳和 URL 輸入
- 🔄 **重複檢查** - 匯入時自動檢查並處理重複案例

## 技術棧

- **React 19** - 前端框架
- **TypeScript** - 類型安全
- **Vite** - 構建工具
- **Tailwind CSS** - 樣式框架
- **Lucide React** - 圖標庫
- **XLSX** - Excel 處理

## 快速開始

### 安裝依賴

```bash
npm install
```

### 環境變數設置

創建 `.env.local` 文件並設置：

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

### 運行開發服務器

```bash
npm run dev
```

應用將在 `http://localhost:3000` 運行。

### 構建生產版本

```bash
npm run build
```

## 主要功能

### 案例管理

- 添加、編輯、刪除案例
- 支援多層級資訊（預覽層、解決方案層）
- 客戶資訊和上線日期管理（僅後台可見）

### 標籤系統

- 三種分類：產業別、業務別、趨勢科技
- 動態標籤管理
- 標籤篩選功能

### 數據匯入/匯出

- **Excel 匯出**：匯出所有案例資料為 Excel 格式
- **Excel 匯入**：批量匯入案例資料
- **重複處理**：自動檢測重複案例並提供處理選項

## 開發說明

### 添加新案例

1. 點擊右上角管理按鈕進入管理介面
2. 選擇「案例資料庫」標籤
3. 點擊「建立項目」
4. 填寫案例資訊並上傳圖片
5. 選擇相關標籤
6. 保存

### 匯入 Excel 資料

1. 在管理介面點擊「匯入 Excel」
2. 選擇 Excel 檔案
3. 如有重複案例，選擇處理方式（跳過/取代/重新命名）
4. 確認處理

### 匯出資料

- **Excel 格式**：點擊「匯出 Excel」下載 Excel 檔案
- **JSON 格式**：點擊「匯出 JSON」下載 JSON 檔案
